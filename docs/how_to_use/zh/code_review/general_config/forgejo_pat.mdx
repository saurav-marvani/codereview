## 简介

要启用 Kody 在 Forgejo 上执行自动代码审查，您需要生成一个**访问令牌**。此令牌允许 Kody 访问您的仓库、拉取请求、Webhook、工单等，具有下面列出的特定权限。

---

## 在 Forgejo 中生成访问令牌

1. 访问您的 Forgejo 实例，点击右上角的头像图标
2. 选择 **Settings**
3. 导航到 **Applications**
4. 在 **Generate New Token** 部分，为令牌命名（例如 `kodus_code_review`）
5. 选择所需的权限（见下文）
6. 点击 **Generate Token**
7. **立即复制**令牌并安全存储 — 关闭页面后将无法再次查看

---

## Kody 所需的权限

令牌必须包含**所有**以下权限才能与 Kody 完全配合使用：

| 类别 | 级别 | 用途 |
|------|------|------|
| `repository` | **读写** | 访问代码、分支、文件，创建 PR 评论，管理 Webhook |
| `issue` | **读写** | 查看和修改工单、标签和里程碑 |
| `notification` | **读取** | 查看通知 |
| `user` | **读取** | 读取基本用户信息 |

---

## 将令牌添加到 Kody

在启用 Forgejo 集成时，将生成的令牌粘贴到 **Kody 集成设置屏幕**中。

---

## 重要说明

- 令牌在创建后**仅显示一次**。请安全保存。
- 如果令牌泄露，请立即撤销并创建新令牌。
- 定期审计令牌及其权限，以确认是否仍然需要。
